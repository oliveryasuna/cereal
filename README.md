# Cereal

A simple binary serialization format for JSON.

Cereal is basically a simplified version of
[PSON](https://github.com/dcodeIO/PSON).
I wrote it to gain a better understanding of how ZigZag encoding works.

While it is (probably) production-ready, there are better alternatives such as
[Protocol Buffers](https://github.com/protocolbuffers/protobuf) and
[PSON](https://github.com/dcodeIO/PSON).
